# /resource-script — Find Resource + Build "Comment for This" Script

Find a genuinely valuable free AI/automation resource, build a viral "comment X to get this" script around it, and create the actual deliverable.

## Steps

### Step 1: Find a valuable resource

Search for something the ICA would actually want. Use WebSearch:
- "free AI automation template [relevant topic]"
- "n8n workflow template free [use case]"
- "AI prompt template [business use case]"
- "free AI business tool [specific task]"
- "ChatGPT prompt pack [business]"

**What makes a good resource:**
- Immediately actionable (not a course or long read)
- Saves real time (30 min+ of work condensed)
- Relevant to coaches/consultants/creators specifically
- Free (no email capture preferred — lowers friction for the CTA)
- Something Zac can credibly "give away" (either a link or something he'll create)

**OR:** Create the resource from scratch — a prompt template, a checklist, a mini-framework, a workflow structure. Zac giving away something he built is more on-brand than just sharing a link.

### Step 2: Validate the resource

Before building the script, confirm:
- Is this genuinely useful (would Zac actually use this)?
- Is it specific enough? ("AI prompts for coaches" = too broad; "5-step discovery call prep prompt" = specific)
- Is the format right for DM delivery? (PDF/doc/link/template)

### Step 3: Create the resource (if building from scratch)

If creating an original resource, build it as a markdown document:
- Title it clearly
- Make it immediately usable (fill-in-the-blank format where possible)
- Keep it under 1-2 pages
- Save to `knowledge/resources/[resource-name].md`

### Step 4: Write the "comment X" script

The script angle: *"I found/built [resource]. It [specific value]. Comment [WORD] and I'll send it to you."*

This is a **TOFU or BOFU** format depending on how broad or niche the topic is.

Structure:
```
[HOOK] — Tease the value without giving it away yet
"There's a free [resource type] that [specific outcome]..."
OR
"I just built a [resource] that [result]. I'm giving it away."

[PROOF] — Why should they trust this?
Brief credibility: "I use this with my clients..." / "This saved me X hours..."

[WHAT IT IS] — Describe it specifically
"It's a [format] that [does X]. You can use it for [specific use case]."

[CTA] — Simple, specific
"Comment '[WORD]' and I'll DM it to you."
OR
"Comment '[WORD]' below and I'll send the link."
```

Caption:
```
Free [resource name] — comment "[WORD]" to get it.

What it includes:
• [Point 1]
• [Point 2]
• [Point 3]

Drop "[WORD]" below 👇
```
